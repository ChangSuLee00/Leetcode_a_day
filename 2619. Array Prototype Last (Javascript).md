# 2619. Array Prototype Last (Javascript)

## Source

https://leetcode.com/problems/array-prototype-last/

## Code

```javascript
Array.prototype.last = function () {
  if (this.length) {
    return this[this.length - 1];
  } else {
    return -1;
  }
};


/**
 * const arr = [1, 2, 3];
 * arr.last(); // 3
 */
```

## How to solve?

리트코드에 자바스크립트 문제가 새로 생겨서 풀어봤다.

this 바인딩을 이용해서 프로토타입의 메서드를 만들어준다.
