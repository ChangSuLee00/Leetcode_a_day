# 67. Add Binary (Javascript)

## Source

https://leetcode.com/problems/add-binary/

## Code

```javascript
/**
 * @param {string} a
 * @param {string} b
 * @return {string}
 */
var addBinary = function (a, b) {
  let sum = [];
  let carry = 0;
  let len_a = a.length - 1;
  let len_b = b.length - 1;

  while (true) {
    let temp = 0;

    if (len_a === -1 && len_b === -1) {
      if (carry) {
        sum.push(carry);
      }
      return sum.reverse().join("");
    }

    if (len_a >= 0) {
      temp += Number(a[len_a]);
      len_a -= 1;
    }
    if (len_b >= 0) {
      temp += Number(b[len_b]);
      len_b -= 1;
    }

    if (temp + carry >= 2) {
      sum.push(carry + temp - 2);
      carry = 1;
    } else {
      sum.push(carry + temp);
      carry = 0;
    }
  }
};
```

## How to solve?

달리 방법이 생각나지 않아 하드코딩 했다...

sum array에 넣고 reverse().join("")을 사용해 순서를 맞춘 뒤 합쳐주었다.
