# 1729. Find Followers Count (MySQL)

## Source

https://leetcode.com/problems/find-followers-count/description/?envType=study-plan-v2&envId=top-sql-50

## Code

```mysql
SELECT
    user_id,
    COUNT(*) AS followers_count
FROM
    Followers f
GROUP BY
    user_id
ORDER BY
    user_id ASC
```

## How to solve?

user_id로 집합 연산을 해준다.
